COMO PUBLICAR NO GITHUB PAGES
1. Extraia este ZIP.
2. Envie index.html, styles.css e a pasta assets para o repositório GitHub.
3. Vá em Settings > Pages.
4. Source: Deploy from a branch.
5. Branch: main.
6. Folder: /(root).
7. Salve.

DOMÍNIO PRÓPRIO
Depois que o site estiver publicado no GitHub Pages:
1. Em Settings > Pages > Custom domain, coloque o domínio.
2. No Registro.br, configure os apontamentos conforme orientação do GitHub Pages.
